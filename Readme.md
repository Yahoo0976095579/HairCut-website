# 金派剪燙染 專業美髮工作室

![金派剪燙染封面](./images/items/manu.png)

## 網站簡介

這是一個專業美髮工作室的形象網站，展現優質的髮型設計服務與作品集。網站採用現代化的設計風格，提供清晰的服務介紹與線上預約功能。

## 主要功能

- 🏠 響應式首頁展示
- 💇‍♂️ 專業服務項目介紹
- 👨‍🎨 設計師資歷與專長展示
- 🖼️ 精美作品集展示
- 📅 線上預約系統
- 📱 完整手機版支援

## 特色服務

- 基礎剪髮：專業修剪技術，精緻造型設計
- 創意染髮：進口高品質染劑，自然顯色
- 質感燙髮：溫塑燙、冷塑燙技術
- 護髮療程：優質護髮產品，深度修復

## 技術實現

- HTML5 / CSS3
- 響應式設計
- CSS Grid / Flexbox 布局
- 現代化動畫效果
- 圖片優化處理
